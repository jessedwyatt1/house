# 09 — Compliance

**Legal, permitting, zoning, and code compliance** for the target jurisdiction.

## Scope

- Zoning: setbacks, max footprint, height, use (residential)
- Building code edition (IRC, IBC, state amendments)
- Energy code (IECC, Title 24, etc.)
- Structural permit and stamped drawings
- Plumbing / mechanical / electrical permits
- Special inspections: insulation, airtightness, engineered glass, water tanks if structural
- Alternative materials or methods (if any)
- Septic/well permits
- HOA / design review if applicable
- Insurance and appraisal considerations (non-standard features)

## High-risk items for this design

| Feature | Typical scrutiny |
|---------|------------------|
| 60' glazed south wall | Energy code, egress glazing, wind load |
| Interior water tanks | Structural, plumbing, moisture, potable if applicable |
| Cantilevered roof | Structural |
| Greywater / rainwater | Health dept, state plumbing code |
| Owner-builder | Which trades still need licensed work |
