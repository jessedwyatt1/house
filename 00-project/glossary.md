# Glossary

| Term | Meaning in this project |
|------|-------------------------|
| **South facade** | The 60' glazed elevation facing south |
| **Building depth** | 25' — distance from south glass to north wall |
| **Tank zone** | ~10' deep area inside the south glass; contains water tanks |
| **Wet wall** | Wall stacking plumbing for kitchen, bath, laundry back-to-back |
| **Thermal mass** | Here: water in tanks, not earth walls |
| **Envelope** | Exterior walls, roof, floor — insulated shell |
| **Overhang** | Cantilevered south roof extension for seasonal shading |
| **Service core** | Kitchen + bath + utility + laundry cluster |
